  @par Example description

  This example provides a short description of how to use the RTC peripheral 
  to exit the MCU from Active-halt mode using the wakeup unit.
  In addition, the example describes how to use the STM8L15x LCD controller to 
  drive the Pacific Display LCD glass mounted on STM8L152x-EVAL board.
